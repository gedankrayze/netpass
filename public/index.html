Core Features

Enterprise Authentication Made Simple

Everything you need to secure your applications with confidence

Military-Grade Encryption

Industry-standard bcrypt hashing with configurable cost factors. Your users' credentials are protected with the same encryption used by government agencies.

Token Management

Stateless JWT-like tokens with automatic expiration, renewal, and revocation. Built for distributed systems and microservices.

Threat Protection

Built-in protection against brute force, credential stuffing, and timing attacks. Real-time threat detection and automated response.

User Lifecycle

Complete user management from registration to deletion. Profile management, password policies, and account recovery flows.

Lightning Fast

Built on Bun runtime for unprecedented performance. Handle thousands of requests per second with minimal latency.

Flexible Storage

Powered by ArangoDB for ultimate flexibility. Supports document and graph data models for complex authorization scenarios.

Developer API

RESTful API Endpoints

Simple, intuitive, and well-documented APIs for seamless integration

POST /api/auth/signup

Register new users with email verification and strong password requirements

POST /api/auth/login

Authenticate users and issue secure access tokens

POST /api/auth/logout

Revoke tokens and terminate active sessions

GET /api/users/profile

Retrieve authenticated user profile information

PATCH /api/users/profile

Update user profile with validation

DELETE /api/users/delete

Permanently delete user account and associated data
